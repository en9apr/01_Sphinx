6.11. LaTeX Equations¶

This is an example of a LaTeX Equation

\begin{equation} \label{eq:aperp}
u(\lambda,T)=\frac{8\pi hc\lambda^{-5}}{e^{hc/\lambda kT}-1},
\end{equation}

Equations are cited like this (~ is a space):

~\ref{eq:aperp}
