6.2.1. Superscripts¶

Code Meaning 
$2x^3$
One character 
$2x^{12}$
More than one character 
$2x^{12^x}$
Power raised to a power

6.2.2. Subscripts¶

Code Meaning 
$x_1$
One character 
$x_{12}$
More than one character 
${{x_1}_2}$
Sub 1, sub 2

6.2.3. Greek Symbols¶

Code Meaning 
$\pi$
Pi symbol 
$\alpha$
Alpha symbol

6.2.4. Trigonometric Functions¶

Code Meaning 
$\sin{x}$
Sine 
$\cos{y}$
Cosine

6.2.5. Log Functions¶

Code Meaning 
$\log_{10}{x}$
Base 10 logarithmic function 
$\ln{y}$
Natural logarithmic function

6.2.6. Square Roots¶

Code Meaning 
$\sqrt{2}$
Square root of 2 
$\sqrt[3]{2}$
Cube root of 2 
$\sqrt{x^2 + y^2}$
Square root of a function 
$\sqrt{1+\sqrt{x}}$
Square root of a square root

6.2.7. Fractions¶

Code Meaning 
$\frac{2}{3}$
Small fraction 2/3 
$\displaystyle{\frac{2}{3}}$
Large fraction 2/3 
$\frac{x}{x^2+x+1}$
Fraction of a function 
$\sqrt{\frac{x}{x^2+x+1}}$
Square root of a fraction
