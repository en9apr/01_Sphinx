5. OpenFOAM Commands¶

Here is a list of some commands in OpenFOAM

  • How to build a mesh?
  • How to run a case?
  • How to post-process data?

5.1. How to build a mesh?¶

Command Meaning 
$ blockMesh
Creates parametric meshes with grading and curved edges

5.2. How to run a case?¶

Command Meaning 
$ icoFoam
Laminar, incompressible, isothermal solver 
$ pisoFoam
Turbulent, incompressible, isothermal solver

5.3. How to post-process data?¶

Command Meaning 
$ paraFoam
Opens Paraview
